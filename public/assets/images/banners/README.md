# Imagens dos Banners

Este diretório contém as imagens dos banners exibidos no dashboard do portal do cliente.

## Imagens necessárias:

1. **vr-va.webp** - Banner para VR e VA (Vale Refeição e Vale Alimentação)
   - Formato recomendado: WebP
   - Dimensões recomendadas: 600x800px ou proporção similar
   - Deve conter o design completo do banner com texto e imagens

2. **totalpass.webp** - Banner para TotalPass
   - Formato recomendado: WebP
   - Dimensões recomendadas: 600x800px ou proporção similar
   - Deve conter o design completo do banner com texto e imagens

## Notas:

- As imagens devem ser otimizadas para web (formato WebP é recomendado)
- As imagens já devem conter todo o texto e design, pois serão exibidas sem overlay adicional
- O banner será clicável e redirecionará para o link configurado em `src/data/banners.js`
